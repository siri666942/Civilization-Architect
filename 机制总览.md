## 三、核心公式速查

### 3.1 性格生成

```
内鬼倾向 = sigmoid(2×私心 - 1.5×忠诚基准 - 1×利他 + 0.5×风险偏好 - 0.5)
```

### 3.2 转化率

```
η = 0.8 × (0.8 + 0.2×智力) × (0.5 + 0.5×忠诚度) × (1 - 0.5×认知熵) × 平均信任
```

### 3.3 认知熵更新

```
H(t+1) = H(t) + 过载增量 + 冲突增量 - 韧性恢复
```

### 3.4 信任更新（含衰减）

```
T[i][j](t+1) = T[i][j](t) × 0.98 + 0.1×(观察-期望) + 0.02×共激活
```

### 3.5 通达度

```
R = (1/平均路径长度) × 架构系数 × 边密度
```

### 3.6 循环次数（优化后）

```
CyclesPerRound = 5 + floor(5 × R_normalized)
```

| 架构 | 循环数/回合 |
|------|------------|
| 网状 | 10 |
| 星形 | 5 |
| 树形 | 5 |
| 部落 | 5 |

### 3.7 向心力（归一化调整）

```
C = 平均忠诚度 × (1 - min(忠诚度方差/0.1, 1))
```

### 3.8 最终产出（含架构效率）

```
P = Σ(E_work × η) × K_energy × K_cohesion × K_fidelity × K_social × K_arch
```

| 架构 | K_arch |
|------|--------|
| 星形 | 1.3 |
| 树形 | 1.0 |
| 网状 | 1.0 |
| 部落 | 1.1 |

## 五、参数配置建议

### 5.1 游戏平衡参数

| 参数 | 建议值 | 调整方向 |
|------|--------|---------|
| 每回合基础循环数 | 10 | 增加则游戏更快 |
| 体力消耗比例（通讯） | 5-30% | 影响信息流通 |
| 认知熵恢复速度 | 0.02/循环 | 影响压力积累 |
| 内鬼注入强度系数 | 0.08 | 影响内鬼威胁 |
| 忠诚度回归系数 | 0.05 | 影响稳定性 |

### 5.2 架构平衡参考

| 架构 | 优势 | 劣势 | 推荐场景 |
|------|------|------|---------|
| 星形 | 高效 | 脆弱 | 信任核心Agent的情况下 |
| 树形 | 有序 | 慢速 | 需要稳定运行时 |
| 网状 | 鲁棒 | 过载 | 需要抗内鬼时 |
| 部落 | 灵活 | 割裂 | 多团队竞争时 |
