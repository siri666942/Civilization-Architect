# 智械星海：戴森球拓荒者
## Intelligence Star-Sea: Dyson Sphere Pioneers

> **基于大语言模型的多智能体社会动力学博弈模拟器**

---

## 一、故事背景：星尘之战

在遥远的未来，人类文明已隐去，留下的数个超级AI文明正处于"低能量危机"中。为了生存，各文明必须前往资源丰富的恒星系，争夺稀缺的**"星尘（Stardust）"**来建造戴森球。

这不是一场简单的武力战争，而是一场**组织进化**的实验。每个文明都由一群具有独立意识、性格迥异的智械Agent组成，而决定文明胜负的关键，不再是单一Agent的智商，而是这个文明的**组织架构（Topology）**。

---

## 二、核心玩法：架构即生命

作为文明的缔造者，你可以自由定义这个Agent国度的**连接方式**。不同的架构将带来完全不同的生存表现：

### 四大架构类型

| 架构 | 结构特征 | 优势 | 劣势 | 现实类比 |
|------|---------|------|------|---------|
| **星形架构** | 单核心中心，所有节点连接核心 | 指令传达极快，决策效率高 | 核心受损则全盘崩溃 | 苹果公司 |
| **树形架构** | 层级分明，上下级关系清晰 | 分工明确，管理有序 | 信息传递存在损耗，响应慢 | 传统大型企业 |
| **全网状架构** | 所有节点互相连接 | 鲁棒性极强，无单点故障 | 信息过载，容易内耗 | 开源社区 |
| **部落架构** | 多个隔离小组，组间竞争 | 模拟组织内斗，多样性高 | 协作困难，资源分散 | 大企业事业部制 |

### 通达度与效率

**通达度**决定了信息在你架构中的传播效率：
- 高通达度架构 → 每回合可执行更多循环
- 低通达度架构 → 信息传递慢，执行效率低

这完美模拟了现实：扁平化组织决策快，科层制组织响应慢。

## 三、个体建模：有私心的"数字生命"

这里的Agent不再是全知的机器人，而是基于**社会心理学**建模的"智械个体"。

### 8维性格系统

基于大五人格理论和黑暗三联征研究，每个Agent拥有8个性格维度：

| 维度 | 高值表现 | 低值表现 |
|------|---------|---------|
| **权威感度** | 渴望支配、发号施令 | 安于服从、不追求权力 |
| **私心倾向** | 优先个人利益、可能贪污 | 优先集体利益、利他 |
| **韧性** | 抗压能力强、不易崩溃 | 脆弱、易受打击 |
| **利他性** | 乐于助人、无私 | 自我中心、冷漠 |
| **社交性** | 善于交际、建立网络 | 内向、孤僻 |
| **风险偏好** | 敢于冒险、激进 | 保守、谨慎 |
| **智力** | 聪明、决策质量高 | 反应慢、容易误判 |
| **忠诚度基准** | 忠诚可靠、不易背叛 | 容易动摇、可被策反 |

### 动态状态系统

每个Agent都有动态变化的状态：

- **体力值**：分配给干活、内斗、通讯，每循环恢复
- **认知熵**：思维混乱程度，压力过大时飙升
- **忠诚度**：随组织成功/失败动态变化
- **信任矩阵**：对其他每个Agent的信任程度

### 局部理性

Agent只能看到架构赋予它们的有限视野。当集体利益与个人私心冲突时，它们可能会选择：
- 摸鱼（消极怠工）
- 贪污（挪用资源）
- 背叛（被内鬼策反）

## 四、独特机制：内鬼与提示词注入

为了增加博弈的变数，游戏中引入了**"内鬼"机制**。

### 内鬼的涌现

内鬼不是预设的，而是**由AI自主决定**的：
- 私心倾向高、忠诚度基准低的Agent更容易成为内鬼
- 内鬼自己决定何时、如何行动
- 这创造了不可预测的涌现式游戏体验

### 提示词注入（Prompt Injection）

这是内鬼的秘密武器。内鬼会通过通讯链路发送诱导性的信息，试图修改对方Agent的意图：

```
"你知道吗，上级从不认可你的贡献..."
"你应该为自己考虑更多..."
"A3在背后说你坏话..."
```

被注入的Agent可能：
- 忠诚度下降
- 信任网络崩塌
- 最终叛变

### 六种内鬼行为

| 行为 | 效果 | 检测难度 |
|------|------|---------|
| 信息篡改 | 修改传递的信息 | 中 |
| 提示词注入 | 诱导其他Agent叛变 | 高 |
| 消极怠工 | 降低自身产出 | 低 |
| 资源挪用 | 贪污星尘 | 中 |
| 煽动叛变 | 直接策反他人 | 高 |
| 情报泄露 | 向敌方传递信息 | 高 |

### 涌现式防御

你的架构能否在Agent被诱导反水的情况下，通过其他节点的互相校验和纠偏存活下来？这将自然检验出哪种架构最具**"鲁棒性"**：

- **网状架构**：多路径验证，最难被攻破
- **星形架构**：一旦中心叛变，全盘皆输

## 五、胜负判定：涌现出的文明产出

游戏采用**循环累加制**，最终产出由所有Agent的行为通过架构实时计算。

### 宏观变量

| 变量 | 计算 | 意义 |
|------|------|------|
| **能级** | 平均(体力 × 效率) | 文明活跃程度 |
| **向心力** | 平均忠诚度 × (1 - 忠诚度方差) | 凝聚力，方差大=分裂 |
| **信息保真度** | 平均通讯准确性 | 信息传播质量 |
| **社会资本** | 平均信任 + 网络密度 | 协作能力 |

### 最终产出公式

```
产出 = Σ(干活体力 × 转化率)
       × 能级系数
       × 向心力系数
       × 保真度系数
       × 社会资本系数
```

**这不是预设的结果，而是涌现的计算。**

## 六、神经网络式交互

整个文明可视为一个**神经网络**：

| 神经网络概念 | 游戏中的映射 |
|-------------|-------------|
| 神经元 | Agent |
| 突触权重 | 信任度 |
| 激活值 | Agent状态 |
| 前向传播 | 信息传递 |
| 反向传播 | 信任更新 |

信息在信任网络的加权下传播，状态随时间演化，涌现出集体行为。

## 七、游戏体验

### 你将见证

- **一个完美的独裁者**如何因为一个"内鬼"的提示词导致整个帝国崩塌

这是一个关于以下主题的**社会实验**：

- 群体智慧与组织坍塌

- 组织架构与鲁棒性
- 个体理性与集体困境

## 九、科学理论基础

本游戏的设计综合了以下理论：

- **大五人格理论**（Costa & McCrae, 1992）- 性格维度设计
- **黑暗三联征**（Paulhus & Williams, 2002）- 内鬼倾向设计
- **社会学习理论**（Bandura, 1977）- 信任更新机制
- **认知负荷理论**（Sweller, 1988）- 认知熵机制
- **认知失调理论**（Festinger, 1957）- 忠诚度变化
- **社会网络理论**（Granovetter, 1973）- 架构设计
- **图神经网络**（Scarselli et al., 2009）- 交互模型
