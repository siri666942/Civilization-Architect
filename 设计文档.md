## 一、项目概述

### 1.1 核心概念
**"架构即生命"** —— 玩家作为文明缔造者，设计AI文明的组织架构（拓扑结构），不同架构将带来完全不同的生存表现。最终目标是通过星尘产出来竞争胜负。

### 1.2 核心体验
- 观察不同组织架构在压力下的涌现行为
- 见证"一个完美的独裁者如何因为一个内鬼的提示词导致帝国崩塌"
- 探索"群体智慧"与"组织坍塌"的社会实验

### 1.3 科学理论基础
- 大五人格理论 + 黑暗三联征
- 社会学习理论
- 认知负荷理论
- 社会网络理论
- 图神经网络
- 动力系统理论

## 三、核心机制摘要

### 3.1 上帝Agent系统
- 生成10个Agent的**8维性格属性**
- 使用Cholesky分解确保属性相关性
- 计算派生的**内鬼倾向**

### 3.2 Agent微观系统
- **体力值**：循环级重置，分配给干活/内斗/通讯
- **认知熵**：反映思维混乱程度，影响决策质量
- **忠诚度**：动态变化，向基准值回归
- **信任矩阵**：10×10矩阵，动态更新

### 3.3 架构与通达度
- 四种架构：星形、树形、网状、部落
- 通达度决定每回合的循环次数
- 不同架构有不同的鲁棒性系数

### 3.4 通讯系统
- **双轨制**：结构化数据 + 自然语言文本
- **信息失真**：信任度×层级×认知熵
- **AI生成**：自然语言由LLM生成

### 3.5 内鬼机制
- **涌现式**：AI自主决定是否成为内鬼
- **六种行为**：篡改、注入、怠工、挪用、煽动、泄露
- **提示词注入**：诱导其他Agent改变忠诚度
- **检测机制**：行为异常、信息矛盾、他人举报

### 3.6 宏观变量与产出
```
P = Σ(E_work × η) × K_energy × K_cohesion × K_fidelity × K_social
```

## 四、核心公式速查

| 公式名称 | 公式 |
|---------|------|
| 内鬼倾向 | `sigmoid(2×私心 - 1.5×忠诚基准 - 1×利他 + 0.5×风险偏好 - 0.5)` |
| 转化率 | `η = 0.8 × (0.8+0.2×智力) × (0.5+0.5×忠诚) × (1-0.5×熵) × 平均信任` |
| 向心力 | `C = 平均忠诚度 × (1 - 忠诚度方差/0.25)` |
| 通达度 | `R = (1/平均路径) × 架构系数 × 边密度` |
| 循环数 | `Cycles = 1 + floor(9 × R_norm)` |
| 最终产出 | `P = Σ(E_work×η) × K_energy × K_cohesion × K_fidelity × K_social` |
| 信任更新 | `T(t+1) = T(t) + 0.1×(观察-期望) + 0.02×共激活` |
| 认知熵更新 | `H(t+1) = H(t) + 过载增量 + 冲突增量 - 韧性恢复` |
