# 智械星海：戴森球拓荒者 - 系统架构文档

> 多智能体社会动力学博弈模拟器后端架构

---

## 1. 项目概述

本项目是一个基于大语言模型(LLM)的多智能体博弈模拟系统，模拟多个文明在戴森球拓荒过程中的社会动力学交互。

### 1.1 核心特性
- **八维性格系统**：基于科学心理学理论设计Agent性格模型
- **四种组织架构**：星形、树形、网状、部落四种拓扑结构
- **内鬼机制**：模拟组织内部的破坏性行为
- **宏观变量计算**：能级、向心力、保真度、社会资本等指标
- **神经网络式交互**：信任矩阵建模社会关系网络

## 4. 模块详解

### 4.1 数据模型层 (models/)

#### models/agent.py

| 类名 | 职责 | 关键属性 |
|------|------|----------|
| `Personality` | 八维性格属性数据类 | authority, selfishness, resilience, altruism, sociability, risk_appetite, intelligence, loyalty_base |
| `AgentState` | Agent动态状态 | energy, cognitive_entropy, loyalty, contribution, efficiency |
| `Agent` | 完整Agent模型 | id, name, personality, state, trust_matrix_row, position, level, centrality |
| `ArchitectureType` | 架构类型枚举 | STAR, TREE, MESH, TRIBAL |
| `TraitorBehavior` | 内鬼行为枚举 | TAMPER, INJECT, SLACK, STEAL, INCITE, LEAK, NORMAL |

**关键方法**:
- `Personality.calculate_traitor_tendency()` - 计算内鬼倾向
- `Agent.initialize_trust()` - 初始化信任矩阵
- `Agent.update_trust()` - 更新信任值（含衰减和Hebbian学习）
- `Agent.calculate_efficiency()` - 计算转化率

#### models/architecture.py

| 类名 | 职责 |
|------|------|
| `ArchitectureConfig` | 架构配置数据类，存储邻接矩阵和各项系数 |
| `ArchitectureFactory` | 工厂类，创建四种预定义架构 |
| `ArchitectureAnalyzer` | 分析器，计算通达度、中心度、路径长度 |

**关键算法**:
- **Floyd-Warshall算法** - 计算最短路径
- **通达度公式** - `R = (1/平均路径) × 架构系数 × 边密度`

### 4.2 核心逻辑层 (core/)

#### core/god_agent.py

| 类名 | 职责 |
|------|------|
| `GodAgent` | 生成Agent性格属性，确保多样性 |

**关键特性**:
- 使用 **Cholesky分解** 生成相关性格属性
- 定义 **8×8维度相关性矩阵**
- 多样性检查确保Agent性格差异

#### core/engine.py

| 类名 | 职责 |
|------|------|
| `GameEngine` | 管理完整游戏流程 |
| `Civilization` | 管理单个文明 |
| `GameState` | 游戏状态数据 |
| `TraitorEvent` | 内鬼事件记录 |

**游戏循环流程**:
1. 重置体力 → 2. 能量分配 → 3. 内鬼行为 → 4. 状态更新 → 5. 产出结算

#### core/macro_variables.py

| 类名 | 职责 |
|------|------|
| `MacroVariableCalculator` | 计算宏观变量 |
| `ProductionCalculator` | 计算产出 |

**宏观变量公式**:
| 变量 | 公式 |
|------|------|
| 能级 | `E = (1/N) × Σ(E_i/100) × η_i` |
| 向心力 | `C = avg_loyalty × (1 - min(Var/0.25, 1)^1.5)` |
| 保真度 | `F = max(0.5, F_base × trust_factor × entropy_factor)` |
| 社会资本 | `S = avg_trust + 0.2 × edge_density` |

## 7. 配置参数

### 7.1 架构参数

| 架构 | 结构系数 | 鲁棒性系数 | 基础保真度 | 效率系数 |
|------|---------|-----------|-----------|---------|
| STAR | 1.5 | 0.1 | 0.9 | 1.3 |
| TREE | 0.8 | 0.3 | 0.7 | 1.0 |
| MESH | 1.0 | 0.9 | 0.85 | 1.0 |
| TRIBAL | 0.6 | 0.6 | 0.6 | 1.1 |
