# 参数定义总表

> 本文档汇总所有游戏机制中的参数，包括性格属性、状态变量、计算参数等。

---

## 一、性格属性参数（固定，来自上帝Agent）

### 1.1 八维性格属性

| 属性名 | 英文名 | 符号 | 范围 | 含义 | 高值表现 |
|--------|--------|------|------|------|---------|
| 权威感度 | Authority | A_auth | [0,1] | 对权力层级的追求 | 渴望支配、发号施令 |
| 私心倾向 | Selfishness | A_self | [0,1] | 个人vs集体利益权衡 | 优先个人利益 |
| 韧性 | Resilience | A_resi | [0,1] | 抗压恢复能力 | 不易崩溃 |
| 利他性 | Altruism | A_altr | [0,1] | 帮助他人倾向 | 乐于助人 |
| 社交性 | Sociability | A_soc | [0,1] | 建立连接能力 | 善于交际 |
| 风险偏好 | RiskAppetite | A_risk | [0,1] | 不确定性决策倾向 | 敢于冒险 |
| 智力 | Intelligence | A_int | [0,1] | 信息处理上限 | 决策质量高 |
| 忠诚度基准 | LoyaltyBase | A_loyal | [0,1] | 初始忠诚程度 | 不易背叛 |

### 1.2 性格相关系数矩阵

```
        A_auth A_self A_resi A_altr A_soc A_risk A_int A_loyal
A_auth   1.0    0.3    0.2   -0.4   0.1   0.3    0.2   -0.2
A_self   0.3    1.0    0.1   -0.6  -0.1   0.2    0.1   -0.5
A_resi   0.2    0.1    1.0    0.2   0.3   0.1    0.3    0.4
A_altr  -0.4   -0.6    0.2    1.0   0.4  -0.1    0.2    0.5
A_soc    0.1   -0.1    0.3    0.4   1.0   0.2    0.1    0.2
A_risk   0.3    0.2    0.1   -0.1   0.2   1.0    0.1   -0.1
A_int    0.2    0.1    0.3    0.2   0.1   0.1    1.0    0.2
A_loyal -0.2   -0.5    0.4    0.5   0.2  -0.1    0.2    1.0
```

### 1.3 派生属性：内鬼倾向

```
TraitorTendency = sigmoid(2.0×A_self - 1.5×A_loyal - 1.0×A_altr + 0.5×A_risk - 0.5)
```

| 内鬼倾向范围 | 分级 | 风险 |
|-------------|------|------|
| [0, 0.2] | 极低 | 安全 |
| (0.2, 0.4] | 低 | 低风险 |
| (0.4, 0.6] | 中等 | 中风险 |
| (0.6, 0.8] | 高 | 高风险 |
| (0.8, 1.0] | 极高 | 危险 |

## 二、动态状态变量

### 2.1 个体状态变量

| 变量名 | 符号 | 范围 | 初始值 | 重置规则 |
|--------|------|------|--------|---------|
| 体力值 | E | [0, 100] | 100 | 每循环重置为100 |
| 认知熵 | H | [0, 1] | 0.1 + 扰动 | 动态变化 |
| 当前忠诚度 | L | [0, 1] | A_loyal | 动态变化 |
| 贡献值累计 | P | [0, +∞) | 0 | 累计增加 |
| 转化率 | η | [0.16, 1.0] | 计算得出 | 动态变化 |
| 心理负担 | MentalLoad | [0, 1] | 计算得出 | 动态变化 |

### 2.2 信任矩阵

| 变量 | 维度 | 范围 | 初始值 |
|------|------|------|--------|
| T[i][j] | 10×10 | [0, 1] | 0.6 + 0.1×(A_soc - 0.5) |
| T[i][i] | 对角线 | 1.0 | 1.0 |

**说明**：初始信任基准提高到0.6，使保真度计算更合理

## 三、体力分配参数

### 3.1 分配边界

| 分配项 | 变量 | 范围 | 说明 |
|--------|------|------|------|
| 干活 | E_work | [20, 80] | 产出星尘 |
| 内斗 | E_conflict | [0, 40] | 权力斗争 |
| 通讯 | E_comm | [5, 30] | 信息传递 |
| 总和 | E_total | ≤ 100 | 体力上限 |

### 3.2 消耗规则

| 行为 | 消耗公式 |
|------|---------|
| 干活 | ΔE = -E_work |
| 发起内斗 | ΔE = -E_conflict |
| 被内斗攻击 | ΔE = -0.3 × E_conflict_received |
| 发送通讯 | ΔE = -E_comm |
| 接收通讯 | ΔE = -0.1 × E_comm_received |

## 四、认知熵参数

### 4.1 增长因素

| 因素 | 增量公式 | 系数 |
|------|---------|------|
| 信息过载 | ΔH += 0.05 × (消息数 - 处理能力) | α_H1 = 0.05 |
| 冲突体验 | ΔH += 0.03 × 内斗次数 | α_H2 = 0.03 |
| 忠诚度撕裂 | ΔH += 0.02 × \|指令 - 意愿\| | α_H3 = 0.02 |
| 被欺骗 | ΔH += 0.08 × 发现次数 | α_H4 = 0.08 |
| 架构层级 | ΔH += 0.01 × depth | α_H5 = 0.01 |

### 4.2 衰减因素

| 因素 | 衰减公式 | 系数 |
|------|---------|------|
| 自然恢复 | ΔH -= 0.02 × (1 - H) | γ_H1 = 0.02 |
| 韧性加成 | ΔH ×= (1 - 0.5 × A_resi) | γ_H2 = 0.5 |
| 同伴支持 | ΔH -= 0.03 × support | γ_H3 = 0.03 |

### 4.3 认知熵影响

| H范围 | 决策质量衰减 |
|-------|-------------|
| < 0.3 | 无影响 |
| 0.3 - 0.6 | -10% |
| 0.6 - 0.8 | -30% |
| ≥ 0.8 | -50% |

**衰减公式**：DecisionQuality = A_int × (1 - 0.5 × H²)

## 五、忠诚度参数

### 5.1 变化因素

| 因素 | 变化公式 | 系数 |
|------|---------|------|
| 组织成功 | ΔL += 0.05 × success | β_L1 = 0.05 |
| 组织失败 | ΔL -= 0.05 × failure | β_L2 = 0.05 |
| 个人不满 | ΔL -= 0.03 × grievance | β_L3 = 0.03 |
| 集体认可 | ΔL += 0.02 × recognition | β_L4 = 0.02 |
| 内鬼诱导 | ΔL -= 0.04 × injection | β_L5 = 0.04 |
| 发现内鬼 | ΔL += 0.02 × discovery | β_L6 = 0.02 |
| 向基准回归 | ΔL = 0.9×L + 0.1×A_loyal | δ_L = 0.1 |

### 5.2 忠诚度阈值效应

| L范围 | 行为倾向 | 转化率因子 |
|-------|---------|-----------|
| > 0.8 | 完全合作 | η_loyal = 1.0 |
| 0.5 - 0.8 | 有所保留 | η_loyal = 0.75 |
| 0.3 - 0.5 | 消极抵抗 | η_loyal = 0.65 |
| ≤ 0.3 | 叛变风险 | η_loyal = 0.5 |

## 六、信任更新参数

### 6.1 更新公式

```
T[i][j](t+1) = T[i][j](t) × (1 - γ_decay) + η_trust × (观察 - 期望) + λ_hebb × 共激活
```

### 6.2 参数值

| 参数 | 符号 | 值 | 说明 |
|------|------|-----|------|
| 衰减率 | γ_decay | 0.02 | 防止无限累积 |
| 学习率 | η_trust | 0.1 | 信任更新速度 |
| Hebbian系数 | λ_hebb | 0.02 | 共激活加强 |
| 正向交互增量 | ΔT+ | 0.1 × (1 - T) | 合作 |
| 负向交互减量 | ΔT- | 0.2 × T | 背叛（双倍惩罚） |
| 社交性加成 | Δα_soc | 0.1 × A_soc | 高社交性学习更快 |

## 七、转化率参数

### 7.1 转化率公式

```
η = η_base × η_int × η_loyal × η_entropy × η_trust
```

### 7.2 各因子

| 因子 | 公式 | 范围 |
|------|------|------|
| 基础值 | η_base = 0.8 | 固定 |
| 智力因子 | η_int = 0.8 + 0.2 × A_int | [0.8, 1.0] |
| 忠诚度因子 | η_loyal = 0.5 + 0.5 × L | [0.5, 1.0] |
| 熵因子 | η_entropy = 1 - 0.5 × H | [0.5, 1.0] |
| 信任因子 | η_trust = Avg(T[connected][self]) | [0, 1.0] |

### 7.3 综合范围

**η范围**：[0.16, 1.0]

## 八、架构参数

### 8.1 架构类型系数

| 架构 | 结构系数 | 鲁棒性系数 | 基础保真度 |
|------|---------|-----------|-----------|
| 星形 | 1.5 | 0.1 | 0.9 |
| 树形 | 0.8 | 0.3 | 0.7 |
| 网状 | 1.0 | 0.9 | 0.85 |
| 部落 | 0.6 | 0.6 | 0.6 |

### 8.2 通达度参数

| 参数 | 公式 |
|------|------|
| 通达度 | R = (1/平均路径长度) × 结构系数 × 边密度 |
| 归一化 | R_norm = R / R_max |
| 循环数 | Cycles = BaseCycles + floor(ExtraCycles × R_norm) |

### 8.3 各架构循环数（N=10，优化后公式）

| 架构 | 平均路径 | R | R_norm | 循环数/回合 |
|------|---------|---|--------|------------|
| 星形 | 2 | 0.15 | 0.15 | 5 + floor(5 × 0.15) = 5 |
| 树形 | 3.2 | 0.05 | 0.05 | 5 + floor(5 × 0.05) = 5 |
| 网状 | 1 | 1.0 | 1.0 | 5 + floor(5 × 1.0) = 10 |
| 部落 | 3.5 | 0.06 | 0.06 | 5 + floor(5 × 0.06) = 5 |

**说明**：BaseCycles=5, ExtraCycles=5（参见评审报告问题1.1修复）

## 九、信息失真参数

### 9.1 失真因子

| 因子 | 公式 | 说明 |
|------|------|------|
| 信任失真 | D_trust = 0.5 + 0.5 × T[j][i] | 低信任=高失真 |
| 关系失真 | D_rel = 1 - 0.1 × \|level(i)-level(j)\| | 跨层级=高失真 |
| 熵失真 | D_entropy = 1 - 0.3 × H_j | 高熵=高失真 |

### 9.2 综合失真

```
Distortion = D_trust × D_rel × D_entropy
```

## 十、宏观变量参数

### 10.1 能级

```
E_level = (1/N) × Σ (E_i/100) × η_i
```

| E_level范围 | 状态 | 效果 |
|------------|------|------|
| > 0.7 | 高度活跃 | 产出+10% |
| 0.5-0.7 | 正常 | 无 |
| 0.3-0.5 | 疲惫 | 产出-10% |
| ≤ 0.3 | 瘫痪 | 产出-30% |

### 10.2 向心力

```
C = L_avg × (1 - min(Var(L)/0.25, 1)^1.5)
```

**说明**：归一化分母调整为0.25（原0.1过于敏感），并使用1.5次幂衰减使曲线更平滑（参见测试诊断修复）

| C范围 | 状态 | 效果 |
|------|------|------|
| > 0.8 | 高度团结 | 协作+15% |
| 0.6-0.8 | 基本团结 | 无 |
| 0.4-0.6 | 分化 | 内耗-10% |
| ≤ 0.4 | 分裂 | 内耗-25% |

### 10.3 产出修正系数

| 系数 | 公式 | 范围 |
|------|------|------|
| K_energy | 0.7 + 0.3 × E_level | [0.7, 1.0] |
| K_cohesion | 0.5 + 0.5 × C | [0.5, 1.0] |
| K_fidelity | 0.8 + 0.2 × F | [0.8, 1.0] |
| K_social | 0.7 + 0.3 × S | [0.7, 1.0] |
| K_arch | 架构效率系数 | [1.0, 1.3] |

### 10.4 架构效率系数

| 架构 | K_arch | 说明 |
|------|--------|------|
| 星形 | 1.3 | 高风险高回报 |
| 树形 | 1.0 | 标准效率 |
| 网状 | 1.0 | 高鲁棒性已是优势 |
| 部落 | 1.1 | 中等难度补偿 |

## 十一、内鬼机制参数

### 11.1 内鬼激活

```
P_activate = TraitorTendency × (1 - L) × OpportunityFactor
```

**机会因子**：
```
OpportunityFactor = 0.3 + 0.2×(1-AvgTrust) + 0.2×H_avg + 0.3×Position_access
```

### 11.2 行为参数

| 行为 | 收益 | 成功率 | 偏好条件 |
|------|------|--------|---------|
| TAMPER | 0.3 | 0.7 | Intelligence高 |
| INJECT | 0.5 | 0.5 | Sociability高 |
| SLACK | 0.1 | 0.9 | 任何 |
| STEAL | 0.4 | 0.6 | Position高 |
| INCITE | 0.6 | 0.4 | Authority高 |
| LEAK | 0.5 | 0.3 | 任何 |

### 11.3 注入效果

```
ΔL = -0.08 × InjectionStrength × (1 - A_resi)
```

**注入强度**：
```
InjectionStrength = 0.5 × T[target][sender] × ContextFit
```

### 11.4 检测概率

| 检测途径 | 基础概率 |
|---------|---------|
| 行为异常 | 0.3 |
| 信息矛盾 | 0.2 |
| 他人举报 | f(信任, 利他性) |
